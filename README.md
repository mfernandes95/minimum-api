# shaw-api

## Usage:

You should:

`npm i` - Run package install.

`npm run dev` - Run the development server.

`npm test` - Run tests.

## Default endpoints:

A `GET` request to `/api/files` you can import one file.csv and save data in database.

A `POST` request to `/api/users` will respond with all users.
